**Canvas WebApp**

- Canvis is a Drawing platform meant to host a large amount of people on the same board using a simple 10 letter url to the Canvis url


- it is built with the following tools

- Frontend
  - Reac(vite)
  - ThreeJS
  - Axios

- Backend
  - Databases
    - Cassandra DB
    - Redis
    - Redis TTL
  - golang mux
  - JWT HS256
  - Apache Kafka

- Cloud
  - Aws Cognito
  - Terraform
  - Aws EKS
